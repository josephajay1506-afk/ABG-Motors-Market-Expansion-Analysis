# ABG Motors Market Expansion-Analysis
Machine Learning and Tableau-based customer purchase prediction and market expansion analysis for ABG Motors.

## Project Overview
This project analyzes customer purchasing behavior using Machine Learning and predicts potential customers for ABG Motors in the Indian market.

A Logistic Regression classification model was built using the Japanese customer dataset and applied to the Indian market dataset to identify potential buyers.

## Key Features
- Data Cleaning and Preprocessing
- Logistic Regression Classification Model
- Model Evaluation and Business Interpretation
- Indian Market Customer Prediction
- Tableau Dashboard Visualization

## Tools and Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Tableau
- Matplotlib
- Seaborn

## Project Results
- Accuracy Score: 68.41%
- Predicted Potential Customers in Indian Market: 54,159
